# 📊 Análisis de Ventas con Python

Este proyecto es parte de mi formación en **Data Analysis**. Aquí exploro un conjunto de datos de ventas de frutas y verduras para entender su comportamiento de precios.

## 🚀 Tecnologías utilizadas
* **Python 3.11**
* **Pandas**: Limpieza y manipulación de datos.
* **Seaborn & Matplotlib**: Visualización de datos y gráficos comparativos.

## 📈 Lo que aprendí en este proyecto
- Cómo limpiar datos (convertir texto a números y arreglar nombres de columnas).
- Crear gráficos de barras agrupados por categorías.
- Automatizar el flujo de trabajo con Git y GitHub desde una MacBook Pro.
